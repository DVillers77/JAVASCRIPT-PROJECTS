CANVAS ELEMENT ASSIGNMENT & CHALLENGE

ASSIGNMENT:

The <canvas> element is a container for graphics that you can use to draw on a webpage using JavaScript. Use HTML to define the canvas demensions and a separate CSS file to give it a border. This is a best practice as it keeps your styling separate from your HTML.

CHALLENGE:

Before you can draw on the <canvas> element, you must use JavaScript to access it. First, a variable is declared using `const` (since it won't change) to get the canvas from the HTML by its ID. Next, a second variable is declared to get the "2D drawing context" using `getContext("2d")`, which provides all the methods needed to draw shapes on the canvas. The final lines of code use the context to define a fill color and then draw a filled rectangle with a specific position and size.
