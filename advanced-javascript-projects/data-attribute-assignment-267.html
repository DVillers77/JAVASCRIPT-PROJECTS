Educational Description

This assignment utilizes the data-* attribute, a powerful HTML5 feature designed to **embed non-visual, custom data directly into HTML elements.** The data acts as a "bridge" that **decouples** the HTML structure (presentation) from the JavaScript logic (behavior). In this project, the data-planet and data-info attributes **contain** the planet's name and its fun fact. This allows JavaScript to read this specific, contextual data using the dataset property (e.g., button.dataset.planet) to dynamically update the page. This method is crucial for building maintainable, interactive user interfaces.
